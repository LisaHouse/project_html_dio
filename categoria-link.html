    • a

    Para redirecionar a link externo, outra página web interna ou levar o usuario á alguma parte específica na mesma página
    Atributos específicos:
    • href: link ou diretório
    • target: _blank(abre em nova aba), _self(abre na mesma página)
    • download: faz download de arquivo fixo
    • target: _blank(abre em nova aba), _self(abre na mesma página)

    a

    disponibilizar na propria página a utilização de outra pagina web(EM HMTL)
    Atributos específicos:
    • src: link
    • width e height: O tamanho que aquela janela ocupará no site
    • name: nome do iframe
    • allow: permissões que aquela iframe pode ter, como uso de camera e autoplay
    • sandbox: Restringe o comportamento do iframe por segurança (evita scripts, redirecionamentos etc.).
      "" (sandbox ativado, bloqueia tudo). "allow-scripts" (permite JavaScript). "allow-same-origin" (permite acesso à mesma origem).
    • loading: Controla o carregamento lazy (adiado) ou eager (imediato).
    • referrerpolicy: Define quais informações de referência (origem) são enviadas ao carregar o iframe.

    img

    Colocar imagens ou gifs na página
    Atributos específicos:
    • src: link ou caminho do arquivo
    • width e height: O tamanho que aquela janela ocupará no site
    • title: Nome que aparece ao passar o mouse na imagem
    • alt: para a acessibilidade, possibilitar a leitura em áudio para com pessoas com deficiencia auditiva

    audio

    Colocar áudio na página
    Atributos específicos:
    • src: link ou caminho do arquivo
    • source: tag filha para indicar o src
    • controls: padrão mostra opção de aumentar áudio, pause etc..
    • autoplay: começa a tocar automático

      • video

      Coloca video na página
      Atributos específicos:
      • src: link ou caminho do arquivo
      • source: tag filha para indicar o src
      • controls: padrão mostra opção de aumentar áudio, pause etc..
      • autoplay: começa a tocar automático
      • track: Recurso de acessibilidade: também possue src para arquivo .vtt ou .srt, onde é colocado as legendas. dentro do track pode se usar a tag kind, onde pode ter as tags, captions, chapters, descriptions, metadata e subtitles, que define a categoria da legenda e srclang que define a lingua
