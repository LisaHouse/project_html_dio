Para textos:

p

Para criar um parágrafo.

mark

Para criar uma marcação no texto, como um marca texto.

small

Para formatação menor do texto.

i

Para formatação para itálico do texto.

strong

Para formatação para negrito do texto.

sub

Para formatação de caracter á frente e abaixo do anterior.

sup

Para formatação de caracter á frente e acima do anterior.

blockquote

Para formatação de texto para inserir uma citação.

abbr

Cria um texto com marcação e ao passar o mouse aparece seu significado

del

Cria um texto com marcação de deletado e mostrando o histórico de alteração com link e data.
