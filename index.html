Estrutura HTML

A estrutura HTML consiste em através de Tags seja possível estruturar um site, sendo um esqueleto. Para ser um código HTML a suas tags base são:
  • html: tag que indica o inicio do HTML
  • head: tag para indicar funções para antes da exibição ao usuário
  • title: tag para mostrar o nome da aba que aparece ao usuário
  • body: tag do corpo do código, tudo que aparecerá ao usuário
