Estrutura HTML

A estrutura HTML consiste em através de Tags seja possível estruturar um site, sendo um esqueleto. Para ser um código HTML a suas tags base são:
  • html: tag que indica o inicio do HTML
  • head: tag para indicar funções para antes da exibição ao usuário
  • title: tag para mostrar o nome da aba que aparece ao usuário
  • body: tag do corpo do código, tudo que aparecerá ao usuário

h1(2,3,4..)

Para criar títulos, quanto menor o número acompanhado ao h maior o título na hieraquia

hr

Para criar uma linha horizontal.

Para textos:

p

Para criar um parágrafo.

mark

Para criar uma marcação no texto, como um marca texto.

small

Para formatação menor do texto.

i

Para formatação para itálico do texto.

strong

Para formatação para negrito do texto.

sub

Para formatação de caracter á frente e abaixo do anterior.

sup

Para formatação de caracter á frente e acima do anterior.

blockquote

Para formatação de texto para inserir uma citação.

abbr

Cria um texto com marcação e ao passar o mouse aparece seu significado

del

Cria um texto com marcação de deletado e mostrando o histórico de alteração com link e data.

Para listas:

ol

Para criação de lista ordenada.

ul

Para criação de lista desordenada.

il

Tag filha para adicionar itens em ol ou ul.

Para links:

a

Para redirecionar a link externo, outra página web interna ou levar o usuario á alguma parte específica na mesma página.
