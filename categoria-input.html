Categoria Input

button

botão simples.
Atributos específicos:
  • value: Define valor de inicialização

checkbox

botão de check, podendo selecionar várias opções
Atributos específicos:

  • checked: Define valor opção inicialização

color(pode ter problema de versão)

seletor de cores.
Atributos específicos:

  • value: Define cor de inicialização
  • list: Mostra opções para escolha

date(pode ter problema de versão)

seletor de datas
Atributos específicos:

  • min: Define data mínima
  • max: Define data máxima
  • step: Intervalo entre datas

datetime-local(pode ter problema de versão)

Seletor de data e hora (sem fuso horário).
Atributos específicos:

  • min: Define data mínima
  • max: Define data máxima
  • step: Intervalo em segundos

email

Campo para e-mail, geralmente com validação automática do navegador
Atributos específicos:

  • multiple: Permite múltiplos emails

file

recebe arquivos
Atributos específicos:

  • accept: Define extensões aceitas(.pdf,.txt...)
  • multiple: permite multiplos arquivos
  • capture: Para celulares permite captura na hora

hidden

campo invisível que pode receber informações mas não fica disponível para o usuário visualizar.

image

Botão de envio representado por uma imagem.
Atributos específicos:

  • src: URL da imagem na internet ou arquivo na pasta do projeto
  • alt: Texto alternativo
  • width/height: Define dimensões da imagem

month(pode ter problema de versão)

Seletor de mês e ano.

number

recebe apenas números
Atributos específicos:

  • min: Define data mínima
  • max: Define data máxima
  • step: Intervalo em meses

password

recebe senhas(*****).
Atributos específicos:

  • minlenght: Define tamanho mínima
  • maxlenght: Define tamanho máximo

radio

Botão de opção única
Atributos específicos:

  • checked: Define opção de inicialização

range

Controle deslizante (slider) para valores numéricos.
Atributos específicos:

  • min: Define data mínima
  • max: Define data máxima
  • step: Intervalo entre valores

reset

Botão para resetar um formulário.
Atributos específicos:

  • value: Define valor de inicialização

search

Campo de busca

submit

botão, que ao ser apertado envia os dados do formulario ao servidor
Atributos específicos:

  • value: Define valor de inicialização

tel

recebe telefones
Atributos específicos:

  • pattern:vaidação de padrão

text

recebe textos de qualquer tipo

time(pode ter problema de versão)

Seletor de horário.
Atributos específios

  • min: Define data mínima
  • max: Define data máxima
  • step: Intervalo em segundos

url

Campo para URLs (validação automática em alguns navegadores).
Atributos específicos:

  • pattern:vaidação personalizada

week(pode ter problema de versão)

Seletor de semana e ano.
Atributos específicos:

  • min: Define data mínima
  • max: Define data máxima
  • step: Intervalo em semanas
